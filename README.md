# Twi Note

This is a desktop application that can accumulate simple memos.  
We aimed for a system similar to Twitter.  
It is useful for recording information that cannot be leaked to the outside world.  

## Environment

- Tauri + Rust
  - Sqlite3
- Vue3 + TS
  - Naive UI
  - xicon

## build

```
$ yarn tauri dev

$ yarn tauri build
```
